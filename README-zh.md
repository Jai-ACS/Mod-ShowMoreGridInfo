# 显示格子更多的信息
手游版的【显示格子更多的信息】 MOD。

此 MOD 移植到手机版后有以下的加强或变化：
1. 支持多种语言
	* 在 MOD 初始版本就支持英文和中文（原本此 MOD 的英文和中文已被融合）
	* 此后，能支持更多语言；你可以以下方法提交新的翻译：
		1. Fork 出此仓库
		2. 增添新语言的翻译文档（[例](Language/OfficialEnglish/codedictionary.txt)）
		3. 在此仓库提交 [Pull Request](https://github.com/Jai-ACS/Mod-ShowMoreGridInfo/pulls)
	* 你也可以在 Discord 平台与我取得联系，请在信息中标记我 (Jai, 用户名：jai.san)
	* 提交成功就会把提交者增加到《鸣谢》章节里（如不是本人翻译的，请说明出自谁的手）
2. 原版本 MOD 是在滑鼠移动时更新格子信息面板的 - 移植版是在格子被选中时更新格子信息面板（和手游版游戏原本没 MOD 时一样）
3. 扩大格子信息面板的面积，由此增加能显示的信息
4. 对一些显示的信息添加颜色，或能增加信息的可读性
5. 门派交易价格已被移除 - 那会在之后成为独立的 MOD

## 主要功能
格子信息面板加强后，显示以下信息：
1. 灵气值
2. 现聚气值和最大聚气值（请阅读下文）
3. 地形与地板
4. 肥沃度（文字描述与数值）
5. 美观（文字描述与数值）
6. 亮度（文字描述与数值）
7. 温度（文字描述与数值）

* 最大聚气值是把全部有效范围能抵达已选格子的聚灵物品的聚灵值加起来而统计出来的。灵气会在5天内聚集在已选格子（最多到最大值），增加格子的灵气值。现有聚气值会在聚气物品被放置或移除时（导致最大聚气值变化）归零。之后聚气值需5天才能回到最大值。

除此之外，在【风水视野】启动时，格子信息面板会转成显示灵气值、各个聚气值、以及五行属性。每个五行元素会有2个数值被显示：
1. 该元素在已选格子的数值
2. 该元素与其他五行元素的百分百比例
* 元素强度值可由这五行的百分百比例计算：
  * （生元素 - 克元素）* 2 + （弟子功法五行元素）

最后，未探索的格子可由此 MOD 显示信息。
